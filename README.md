# All-in-One WP Migration Unlimited Extension

This is an **unofficial extension** for the WordPress plugin [All-in-One WP Migration](https://wordpress.org/plugins/all-in-one-wp-migration/) that allows you to bypass the upload size limit. With this extension, you can import/export WordPress sites of **any size**.

> ⚠️ **Disclaimer:** This is a third-party extension. Use it at your own risk. Make sure to backup your site before using it.

---

## Features

- Remove the default 512MB / 2GB upload size limit.
- Import/export WordPress sites of any size.
- Fully compatible with All-in-One WP Migration plugin.
- Simple installation process.

---

## Installation

1. Download the extension zip file.
2. Log in to your WordPress admin panel.
3. Go to **Plugins → Add New → Upload Plugin**.
4. Upload the zip file and click **Install Now**.
5. Activate the plugin.

---

## Usage

1. Navigate to **All-in-One WP Migration → Import / Export**.
2. You will now be able to import/export files **without size restrictions**.
3. Follow the standard All-in-One WP Migration workflow.

---

## Notes

- Make sure your server has enough resources (memory, disk space) to handle large site imports/exports.
- This extension is intended for **developers and advanced users**.
- If you encounter any issues, check your server PHP settings (`memory_limit`, `max_execution_time`, `upload_max_filesize`, etc.).
